# ADHDay

문서 버전: 1.1

작성일: 2025년 6월 25일

프로젝트명: **ADHDay**

프로젝트 개요 : **ADHDay**는 ADHD인의 증상과 감정 상태, 약물 복용 효과를 기록하고, 주치의 상담 및 치료 효과 분석을 돕는 어플리케이션이다.
인지행동치료(CBT) 기반의 생산성 기능과 집중 도구를 함께 제공하여, 사용자가 자신의 패턴을 파악하고, 일상을 스스로 개선해나갈 수 있도록 지원한다. 
ADHD는 단지 '부족함'이 아닌 '다르게 작동하는 뇌'에서 비롯된 것이다. ADHDay는 사용자가 그 차이를 이해하고, 고통은 덜고 가능성은 확장할 수 있도록 돕는 데 목적이 있다.
